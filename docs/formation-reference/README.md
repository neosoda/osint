# Formation de référence — OSINT & Ingénierie Sociale

Ce dossier transforme le contenu actuel du dépôt en **parcours de formation professionnel**, orienté exécution terrain et gouvernance sécurité.

## Structure idéale proposée

- `00-etat-actuel.md` : audit initial (forces, lacunes, risques).
- `01-plan-amelioration.md` : plan d’amélioration **fichier par fichier**.
- `modules/` : contenu pédagogique complet prêt à déployer.
- `guides/` : templates opérationnels (labs, évaluation, qualité).

## Principes de conception pédagogique

1. **Théorie → Procédure → Hands-on → Évaluation** dans chaque module.
2. **OPSEC by design** : chaque TP contient périmètre autorisé, interdits, journalisation.
3. **Reproductibilité** : commandes explicites, livrables attendus, critères de succès.
4. **Compatibilité production** : pas de dépendance à des démonstrations fragiles non contrôlées.

## Parcours cible

1. M01 Fondamentaux OSINT
2. M02 SOCMINT, GEOINT, Métadonnées, Breach Data
3. M03 OSINT technique & infrastructure
4. M04 Ingénierie sociale (psychologie, biais, pretexting)
5. M05 Phishing/Vishing + stratégie de défense
6. M06 Outils modernes (TruffleHog, EyeWitness, dnscat2)
