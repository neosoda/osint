## Pretexting — méthode
1. Définir objectif mesurable.
2. Construire identité de couverture cohérente.
3. Préparer script principal + variantes d’échec.
4. Préparer “safe exit” sans pression abusive.

```text
Exemple de structure de script
- Ouverture (contexte plausible)
- Validation minimale (sans collecte sensible)
- Demande contrôlée (objectif)
- Sortie (remerciement, clôture)
```
